# Systems Analysis & Design Toolbox

## Overview
This repository contains a systems analysis and design toolbox created as part of my MIS 3373 coursework. The toolbox brings together key concepts, frameworks, and examples used to evaluate, plan, design, and implement information systems.

## Topics Covered
- Systems Development Life Cycle
- Project planning and scheduling
- Gantt charts and PERT estimation
- Agile and Scrum
- Build vs. buy software decisions
- Project feasibility analysis
- Stakeholder analysis
- Business process alignment
- Security and compliance considerations

## Purpose
The purpose of this toolbox is to show my understanding of how information systems are evaluated and designed from both a business and technical perspective.

## Skills Demonstrated
- Systems analysis
- Business process analysis
- Requirements thinking
- Project planning
- Documentation
- Feasibility assessment
- Technology decision-making
